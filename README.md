# Barnabas Dental

Dental clinic management system — Django REST API and React (Vite) frontend.

## Project layout

| Component | Path |
|-----------|------|
| Backend (Django) | [`backend/`](backend/) |
| Frontend (React) | [`frontend/`](frontend/) |

Render and Vercel deploy from these **repo-root** directories. A legacy copy may exist under `barnabas-dental-system/` for local reference; it is not used for production deploys.
